# BeeBoard v0.1 Simulation Lab

Локальный Python-стенд для платы BeeBoard v0.1.

Что есть:

- измерение расчетных напряжений, токов и мощности по линиям питания;
- быстрые режимы нагрузки FPGA, LiFi, IMU, крыльев, пружины и дрели;
- вкладка `3D Board Review`, которая открывает GLB-модель платы из KiCad-проекта;
- API `/api/simulate` и `/api/health` для автоматических проверок.

Запуск:

```powershell
.\.venv\Scripts\python.exe -m uvicorn app:app --host 127.0.0.1 --port 8765
```

Открыть:

```text
http://127.0.0.1:8765
```

Важно: модель электрической цепи здесь является board-level оценкой для архитектуры и бюджета питания. Это не финальная SPICE/IBIS-симуляция конкретных компонентов.

Bee Linked Simulation - финальная версия для сдачи

Название проекта:
Bee Linked Simulation: 2D Bee Face Patrol with Ursina 3D Visualization

Краткое описание:
Это игровая программа на Python, в которой основная игра работает на 2D-карте, а 3D-сцена Ursina используется как синхронизированная визуализация. Игрок управляет пчелой, собирает золотые точки энергии, избегает полей зрения трех движущихся персонажей и запускает локальное NVIDIA CUDA-распознавание лиц через PyTorch.

Цель игры:
Найти и распознать всех трех персонажей: Adi, Faraj и Slava. После распознавания всех лиц игра выводит сообщение о победе: Mission Complete! You win. All faces detected.

Основные правила:
1. Пчела двигается по 2D-карте.
2. Золотые светящиеся точки пополняют энергию.
3. Энергия постепенно заканчивается.
4. Если энергия закончилась, игра завершается поражением.
5. У каждого лица есть красная зона обзора.
6. Если пчела попадает в зону обзора, игра завершается поражением.
7. Распознавание лиц выполняется локально на NVIDIA CUDA через Python и PyTorch.
8. При отсутствии CUDA система явно сообщает об ошибке и не переходит на CPU.

Уровни сложности:
Easy - медленные лица, больше еды, медленный расход энергии.
Normal - стандартный баланс скорости, еды и энергии.
Hard - лица быстрее, энергии меньше, играть сложнее.

Управление:
W / Up - двигаться вперед.
S / Down - двигаться назад.
A / Left - повернуть влево.
D / Right - повернуть вправо.
C - выполнить NVIDIA CUDA-распознавание лица.
R - перезапустить уровень.
M - вернуться к выбору сложности после конца игры.
Sound - включить или выключить звуковые эффекты.
Stop / Esc - выйти из игры.

Структура проекта:
Start Bee Linked Game.bat - запуск игры.
Start Linked Bee Experience.py - стартовое окно и запуск связанной 2D/3D-системы.
bee_face_patrol.py - основная 2D-игра на Pygame.
Bee_3D_Standalone/main.py - 3D-визуализация на Ursina.
Bee_3D_Standalone/bee_space_control.json - файл синхронизации между 2D и 3D.
Bee_3D_Standalone/identity_matcher.py - локальное NVIDIA CUDA-распознавание лиц.
Bee_3D_Standalone/local_face_ai - локальные данные и эталоны лиц.
Фоновая музыка отключена и не входит в опубликованный установочный архив.
Bee_3D_Standalone/drone model - 3D-модели пчелы и персонажей.
GAME_DOCUMENTATION_RU.md - подробная документация проекта.

Принцип связи 2D и 3D:
Главной является 2D-карта. Она управляет положением пчелы, лиц и точек энергии. Эти данные записываются в JSON-файл синхронизации. 3D-сцена читает этот файл и показывает ту же ситуацию в трехмерном виде. Поэтому 2D-игра управляет 3D-визуализацией.

Распознавание лиц:
Распознавание выполняется локально на Python при нажатии C. Программа сравнивает данные с эталонными изображениями Adi, Faraj и Slava и показывает, кто обнаружен, сколько заняла обработка и когда она была выполнена.

Что демонстрирует проект:
- объектно-ориентированную структуру программы;
- игровую логику с состояниями победы и поражения;
- уровни сложности;
- управление персонажем;
- звук и фоновую музыку;
- 2D-графику на Pygame;
- 3D-визуализацию на Ursina;
- обмен данными между разными частями программы;
- локальное распознавание лиц на NVIDIA CUDA;
- готовую структуру проекта для запуска и проверки.

Как запустить:
Открыть файл Start Bee Linked Game.bat и выбрать уровень сложности в стартовом окне.
